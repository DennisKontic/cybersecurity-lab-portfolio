# Dennis Kontic Cybersecurity Lab Portfolio

Welcome to my cybersecurity lab portfolio. This repository documents my hands-on technical projects in blue team security, SOC analysis, digital forensics, incident response, detection engineering, threat hunting, malware analysis, and web application security.

The goal of this portfolio is to show practical cybersecurity skills beyond certifications by documenting how I investigate alerts, analyze attacker behavior, map activity to frameworks like MITRE ATT&CK, write detection logic, and communicate findings clearly.

All projects in this repository are based on authorized lab environments, personal research, public training platforms, or sanitized professional experience. No confidential employer, customer, or production data is included.
